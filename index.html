Aprendendo git com Hora de Codar

COMANDOS GIT

  • git init
  • git add README.md
  • git commit -m "first commit"
  • git branch -M main
  • git remote add origin https://github.com/dlemos09/...
  • git push -u origin main

Caso tenha muitos arquivos para serem adicionados de uma só vez, utilize o comando git add .

Após realizar o git add ou alterar um código, é importante realizar o "git commit" para salvar o documento no servidor. Lembre-se, cada commit é fundamental para informar o que foi alterado.

Caso queira realizar o commit a mais de um arquivo, utilize o comando git commit -a -m ou use apenas o git commit -m se for para apenas um código

Caso queira "puxar" o arquivo do github para seu servidor locar, utilize o comando git pull

O ato de baixar um repositório de um servidor remoto é chamado de clonar repositório, utilize o comando git clone. Lembrando que não é usado com muita frequência, apenas quando estamos entrando em um novo projeto.

Para remover um documento ou uma linha de código com o git, utilize o comando git rm.

Para visualizar todo o histórico do seu código, utilize o comando git log.

Para renomear ou mover um arquivo com git, utilize o comando git mv

O arquivo modificado pode ser retornado para o estado original, utilize o comando git checkout

Uma técnica muito utilizada é ignorar arquivos do projeto, para isso, devemos inserir um arquivo chamado .gitignore na raiz do projeto. Nele podemos inserir todos os arquivos que não devem entrar no versionamento. Isso é útil para arquivos gerados automaticamente ou arquivos que contêm informações sensíveis

Com o comando git reset podemos resetar as mudanças feitas. Geralmente é utilizado com a flag --hard Todas as alterações commitadas e também as pendentes serão excluídas;

Manipulando branchs

Para visualizar uma branch, utilize o comando, git branch.

Para criar uma branch, utilize o comando, git branch "nome da branch".

Para deletar uma branch, utilize o comando git branch -d Atenção !! NÃO É COMUM DELETAR BRANCHS

Para mudar de branch, utilize o comando git checkout. Mas, se quiser criar uma branch e mudar ao mesmo tempo, use o comando git checkout -b "nome da branch"

Para unir duas branchs, use o comando git merge
